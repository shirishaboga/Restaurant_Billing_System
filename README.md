# Restaurant Billing System (C)

## Description
This is a simple restaurant billing system developed in C language.
It calculates the total bill based on user input.

## Features
- Menu-driven program
- Calculates total amount
- User-friendly interface

## Language Used
C

## How to Run
1. Compile the program using a C compiler
2. Run the executable file
3. Enter food items and quantity
